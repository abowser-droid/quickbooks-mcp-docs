QuickBooks MCP Server

A Model Context Protocol (MCP) server that connects Claude Desktop to QuickBooks Online.

About

This application enables Claude Desktop to securely access your QuickBooks Online data for accounting queries and reports.

Features

  • List and search customers
  • View invoices filtered by status, date, or customer
  • Check account balances
  • Generate P&L, Balance Sheet, and A/R Aging reports
  • Search transactions
